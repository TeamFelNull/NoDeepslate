# Deepslate to Stone Mod (Minecraft 1.20.1 / Forge)

深層岩（Deepslate）をすべて通常の石に置換するModです。  
1.12.2以前のように、地下がどこまでも石だけになります。

---

## 動作内容

チャンクがロードされる際に、Y=-64〜Y=0 の範囲にある深層岩系ブロックを
対応する石系ブロックに自動置換します。

| 置換前（深層岩系）         | 置換後（石系）       |
|--------------------------|------------------|
| 深層岩                    | 石               |
| 砂利状の深層岩             | 丸石             |
| 深層岩 石炭鉱石            | 石炭鉱石         |
| 深層岩 鉄鉱石              | 鉄鉱石           |
| 深層岩 銅鉱石              | 銅鉱石           |
| 深層岩 金鉱石              | 金鉱石           |
| 深層岩 レッドストーン鉱石   | レッドストーン鉱石 |
| 深層岩 エメラルド鉱石       | エメラルド鉱石   |
| 深層岩 ラピスラズリ鉱石     | ラピスラズリ鉱石 |
| 深層岩 ダイヤモンド鉱石     | ダイヤモンド鉱石 |

### Y座標について
- **最低Y: -64** のまま（バニラ1.20.1と同じ）
- **Y=0 以下**がすべて石になります（深層岩が生成される範囲のみ処理）


## 注意事項

- **新規ワールド推奨**: 既存ワールドでも動作しますが、未ロードのチャンクのみ変換されます。すでにロード済みのチャンクは変換されません。
- サーバー環境でも動作します（サーバー側でのみ処理されます）。
